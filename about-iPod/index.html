iPod 相关的介绍/导购

本文介绍的范围，是关于 iPod 的各类机型（iPod “Classic”/mini/nano/shuffle 等，因为 Touch 我还没玩过，所以暂且不表）以及 iPod 的一些配件等。并依据我的经验，做个简单地导购指南，希望能对想玩 iPod 的你提供一些帮助。

iPod “Classic” 系列

这里泛指传统的 iPod 系列（iPod 4/“Video”/Classic 等），而非「iPod Classic」，此系列的优点如下：

  • 大容量存储。此系列使用 1.8 英寸机械硬盘作为存储，因此能拥有比较大的容量。
  • 易于改装，可玩性好。（改大容量电池、改 SSD 等）

劣势：

  • 机械硬盘较易损坏。（改为 Flash 存储解决）
  • 由于使用机械硬盘，时常出现跳歌，系统卡死等情况。（改为 Flash 存储解决大部分）

此系列是大家公认的「最经典」的 iPod，可玩性很高，甚至改装后能媲美一些现代设备（如 iPod Classic 改装 1TB 存储、3000 毫安大电池（不断播放音乐能播放三天））。

此处又可以分为三种，一种是「黑白屏」iPod（iPod 4 以及之前），iPod 3 之前的早期 iPod 还具有 Fireware 支持，黑白屏的 iPod 看起来更加的 old school。

一类是 iPod “Video”（即 iPod 5th/5.5th）被大家热情的称之为「Video」是因为从这一代开始终于可以播放视频了。它也是大家公认的「声音最好的 iPod」，它的可玩性也最好，没有后来 iPod Classic 系列繁多的限制，这款机型也是本文章最推荐的。

一类是 iPod 6th 以后（iPod Classic），它的限制繁多，但是它的好处是官方系统支持 LBA 寻址（大于 137GB 的空间），因此它可以在不刷 Rockbox 的情况下支持大于 137GB 的存储空间（可改装 1TB 空间）。

关于此系列的改装，这里所有的机型都可以改装 Flash 存储和大电池，但对其硬件的挑选稍有要求。

而对于存储方案的改装，有多种选择：

  • SSD -> ZIP
  • CF -> ZIP
  • microSD -> SD -> CF -> ZIP

关于这个问题，microSD -> SD -> CF -> 1.8 ZIP 是最好的选择。因为 SSD 比较耗电，且适合 iPod 的 SSD 贵、且 iPod 非常挑硬盘。（似乎只有金胜维的几款是大家所熟知能用的，而金胜维的品质大家都知道……）

而 CF 卡则是单纯的比较贵。

所以最后选择第三种方案最好，当然，第三种方案也用两种方案。

  • iFlash：质量好，他们有一个配件可以四张 SD 卡组 RAID（1TB 存储就是这么改装来的），但是贵。
  • 华强北无印牌方案：比较便宜，但出不出问题得看脸（一般不会？）

关于 LBA 寻址的问题，主要出现在随机播放上，所以如果你使用 iPod Classic(7th) 之前的机型，且改装的容量大于 137GB，请使用 Rockbox。
